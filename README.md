## 🧠 Overview

**Silhouette FX** is a dedicated post-production application designed for professional rotoscoping, digital paint, compositing, image restoration, and motion-based visual effects. It provides artists with an extensive collection of tools for creating production-quality masks, repairing footage, removing unwanted objects, and preparing complex shots for compositing pipelines.

The software combines advanced spline editing, GPU-accelerated image processing, motion analysis, tracking technology, and a flexible node-based workflow that allows visual effects teams to manage demanding film and television projects with greater precision and efficiency.

Designed for VFX compositors, roto artists, cleanup specialists, film editors, animation studios, and Windows users.

## 🚀 Key Features

| Feature | Description |
|----------|-------------|
| 🎬 **Professional Rotoscoping Workflow** | Build highly accurate animated masks using multiple spline systems for complex object isolation and compositing tasks. |
| 🖍️ **Digital Paint & Cleanup Tools** | Remove production equipment, tracking markers, logos, blemishes, reflections, wires, and other unwanted elements frame by frame. |
| 📍 **Integrated Motion Analysis** | Utilize advanced tracking technologies to attach masks and visual effects accurately to moving objects and camera motion. |
| 🧩 **Node-Based Effects Pipeline** | Construct flexible compositing workflows with interconnected processing nodes for non-destructive editing and visual effects creation. |
| 🌈 **Keying & Matte Refinement** | Generate clean alpha channels, refine transparency, isolate fine details, and prepare footage for advanced compositing. |
| 🌀 **Image Warping & Morphing** | Apply controlled geometric deformations, facial adjustments, transitions, and animation effects using precision editing tools. |
| ⚡ **Optimized GPU Performance** | Process high-resolution footage efficiently with hardware acceleration for responsive editing and faster rendering. |
| 🔗 **Pipeline Compatibility** | Exchange projects with industry-standard editing, finishing, and compositing applications through flexible workflow integration. |
